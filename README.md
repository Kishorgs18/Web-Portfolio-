# Photography Portfolio — Responsive Front-End

A responsive **photography portfolio website** built from scratch with
**HTML, CSS, and JavaScript**, focused on clean layout, dynamic visual
variations, and viewport-aware responsive design.

🔗 **Live demo:** enable GitHub Pages on this repo (Settings → Pages → deploy from
`main`/root) to publish it at `https://kishorgs18.github.io/Web-Portfolio-/`.

## Features

- Responsive layout that adapts across desktop, tablet, and mobile viewports.
- Dynamic visual variations driven by JavaScript.
- Hand-written CSS — grid/flex layouts, spacing, and viewport manipulation.

## Built with

HTML5 · CSS3 · JavaScript (vanilla)

## Files

| File | Role |
|---|---|
| `index.html` | Page structure |
| `style.css` | Styling & responsive layout |
| `script.js` | Interactivity / dynamic variations |
